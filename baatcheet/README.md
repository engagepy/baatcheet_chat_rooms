# Django BaatCheet

## A Scafold Project Enabling CRUD functionality:

- Chat Rooms
- Login
- Sessions 
- Activity
- S3 Profile Images 
- RDS Postgres db

### Follow these steps:

> Activate virtual environment 

> Close the Repo

> Install 'requirements.txt'

```pip3 install -r requirements.txt```

> cd into the 'baatcheet' directory 

> Run ```manage.py```

```python3 manage.py runserver```
